## 🧠 Contenido 
Este trabajo contiene una GIU con una implementación de CRUD en python usando la librería Tkinter. El programa contiene un formulario para gestionar usuarios con:
- Nombre 
- Correo electrónico
Y 4 botones: Create, Read, Update, Delete
